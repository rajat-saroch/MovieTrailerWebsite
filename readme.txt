Movie Trailer Website :
---------------------
this is a server side code written in python to experience enjoying your favourite movies trailers.


Steps to follow :
---------------
1. download zip file
2. run fresh_tomatoes.html
3. enjoy the site


Want to add your favourite movies??
---------------------------------
1. download python
2. edit entertainment_center.py using python IDE
3. create your Movie object like
   object_name = media.Movie('Movie_name', 'Movie_storyline', 'poster_url', 'trailer_url')
4. add your created objects in array
   movies = [object_name]
5. run the program by using F5
